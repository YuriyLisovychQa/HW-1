Cheat sheet for Git

Comands What comands are doing
git config --global user.name Blabla Bla The username are associated with commits on all repositories
git config --global user.email blabla@bla The email are associated with commits on all repositories
git init Create new repository or reinitalize an existing one
git status Show the working tree status. List all new or changed files
git add Add file content to the files
git commit Record changes to the repository
git log Shows the list commits
git checkout Change the branch on the project
git branch Creating new branch
git push update remote repository with local
git pull Combine changes from the remote repository into branch
git merge Combine two or more development histories together
git clone Clone a repository to new directory

.gitignore file Needed to hide files and folders from the Git version control system.
